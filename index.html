About

jMini is a JavaScript toolbox for lightweight and efficient web development. It is aimed at supporting standards-compliant coding practices, and helps you to keep download size and general overhead minimal.

Download

While you can simply download the entire jMini library from GitHub, this site features a download tool that lets you tailor the toolbox to your needs, i.e. only download the functionalities that you actually need.

Status

jMini is in “under development” status. There may be errors and things may still change. Make sure to follow up on the development if you are using it for your projects.
